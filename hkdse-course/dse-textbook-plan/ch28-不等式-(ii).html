Ch28 不等式 (II)

前置：Ch20 直线方程

模块目标： 在 Ch11 一元一次不等式的基础上，将不等式理论推向三个方向：(1) 引入逻辑连词“且”与“或”，将多个一元不等式组合成不等式组，并以集合交/并运算处理解集；(2) 将一元二次函数的图像知识（Ch21）应用于一元二次不等式的求解——以“以形助数”的图解法与因式分解/判别式的代数法并行；(3) 将不等式的视野从一维数轴拓展至二维坐标平面，以二元一次不等式表示半平面区域，进而以线性规划求解现实中的最优化问题。

28.1 一元不等式组 (Systems of Inequalities in One Variable)

本节建立不等式组的逻辑基础——“且”对应集合交集、“或”对应集合并集——并以此为框架，系统处理一元一次不等式组与一元二次不等式的求解。

28.2 联立二元一次不等式 (Systems of Linear Inequalities in Two Variables)

本节将不等式的讨论从一维数轴拓展至二维坐标平面。一条二元一次不等式对应一个半平面；多条不等式的联立则定义了这些半平面的交集——即可行域。

  • 定理 28.6 (可行域定理 / Feasible Region Theorem) 二元一次不等式组的解区域为各不等式所对应半平面的交集： 此交集称为此不等式组的可行域 (Feasible Region)。

  • 定理 28.7 (可行域的形状 / Shape of the Feasible Region) 由有限个二元一次不等式定义的可行域为凸多边形区域（Convex Polygonal Region），可能为有限（闭凸多边形）或无限（无界凸多边形区域）。

    说明： 每个二元一次不等式定义一个半平面（凸集）。凸集的交集仍为凸集。由若干半平面的交集所得之凸集为凸多面体（在二维即凸多边形区域）。DSE 阶段无需给出严格证明，理解几何直观即可——若干条直线像“刀”一样切割平面，留下的公共部分必为由直线段围成的多边形区域。

  • 方法 28.8 (画出可行域的步骤 / Steps for Sketching a Feasible Region)

    1. 画各边界直线： 对每条不等式，画出其对应的边界直线（实线或虚线取决于是否含等号）。
    2. 定各半平面： 以选点测试法（方法 28.7）确定每条不等式对应的半平面。
    3. 取交集： 在各不等式对应的半平面内以箭头或阴影标记，所有标记重叠的区域即为可行域。
    4. 标注： 在可行域内标注“”或打上斜线，边界的实线/虚线状态须清楚可辨。

    Remark： DSE 作图题通常要求将可行域以斜线或阴影清楚标出。若边界为虚线，表示该边界不属于可行域，需特别留意。

28.3 线性规划 (Linear Programming)

线性规划是二元一次不等式组的应用延伸——在可行域的限制下，求一个二元一次表达式（目标函数）的最大值或最小值。这是 DSE 中少数需要综合运用代数、几何与逻辑推理的课题，也是现实中资源分配、生产排程等最优化问题的数学模型。

  • 定义 28.6 (线性规划问题 / Linear Programming Problem) 在给定的约束条件 (Constraints)（由若干二元一次不等式组成的系统）下，求目标函数 (Objective Function) （ 为常数，不同时为 ）的最大值或最小值。

    标准形式：

  • 定义 28.7 (可行解与最优解 / Feasible Solution & Optimal Solution)

    • 可行解 (Feasible Solution)： 满足所有约束条件的 。
    • 最优解 (Optimal Solution)： 在所有可行解中，使目标函数取得最大值（或最小值）的 。

  • 定理 28.8 (顶点原理 / Vertex Principle) 若线性规划问题的可行域为非空的有界凸多边形（闭区域），且目标函数 在此可行域上可取得最大值（或最小值），则最值必在可行域的某个顶点处取得。

    证明思路（几何直观）： 目标函数 可改写为 （设 ）。固定 时，这是一条斜率为 的直线；不同 值对应此直线族中的不同直线—— 越大，直线的 截距越大（若 ）。将此直线像一把“尺子”沿可行域的方向平移：当直线首次接触可行域时 最小，最后脱离可行域时 最大。由于可行域为凸多边形，首次及最后接触点必为多边形的顶点。

    Remark： 若最优解出现在整条边界上（例如目标函数的直线斜率恰好与可行域某一边界平行），则该边界上的所有点均为最优解。但该边界的端点（即顶点）亦是可行解，代入目标函数可得相同最值。因此，仅检查顶点即已足够。

    Remark： 对于无界可行域，最大值或最小值可能不存在（目标函数可无限增大或减小）。此时需结合几何直观判断。

  • 方法 28.9 (线性规划的标准解题步骤 / Standard Procedure for Linear Programming)

    1. 设变量： 明确 、 分别代表什么（常见为两种产品的产量）。
    2. 列约束： 将题目中的所有限制条件（资源限制、非负条件 等）转化为二元一次不等式。
    3. 写目标： 将题目要求的最大化/最小化量写成 的形式。
    4. 画可行域： 在坐标平面上绘出所有约束条件对应的半平面，标出可行域 （方法 28.8）。
    5. 求顶点： 计算可行域各顶点的坐标。顶点为两条边界直线的交点——解该两直线的联立方程即得。若可行域为三角形，需求三个顶点；为四边形则需求四个。
    6. 代目标： 将各顶点坐标代入 。
    7. 比较得结论： 比较各顶点的目标函数值，取最大值或最小值。以文字写出最终答案（例如“当 时， 的最大值为 ”）。

    Remark： DSE 线性规划应用题通常要求以整数为答案（因为实际问题中产量等不可为分数）。若最优顶点的坐标非整数，需检查邻近的整数格点是否仍在可行域内，选取使目标函数最优的整数点。

  • 推论 28.3 (目标函数的几何意义) 目标函数 的值对应一直线族的 截距（以 表示，设 ）。因此：

    • 求 的最大值 在可行域内，将斜率为 的直线尽可能向 截距增大的方向平移，求最后一个接触点。
    • 求 的最小值 在可行域内，将斜率为 的直线尽可能向 截距减小的方向平移，求第一个接触点。

    Remark： 此几何观点有助于检查答案的合理性——若目标函数直线的斜率介于可行域某两条边界直线的斜率之间，则最优解必在某一顶点；若斜率与某一边界平行，则该边界上所有点均可为最优解。

第二十八章总结

  1. 不等式组的逻辑基础（定理 28.1–28.2）： “且”对应集合交集 、“或”对应集合并集 。一元不等式组（以“且”联结）的解集为各不等式解集的交集。数轴法（方法 28.1）与口诀法（方法 28.2）是求解一元一次不等式组的两种互补工具——前者直观，后者快捷。

  2. 一元二次不等式（定理 28.3–28.4）： 有两条求解路径。图解法利用 Ch21 中二次函数抛物线的图像——由 及 的符号决定 何处为正、何处为负。代数法将二次式因式分解后进行符号分析，本质上是定理 28.2 的应用——二次不等式的解为若干区间的并集。 时二次三项式符号恒定（推论 28.1）。

  3. 二元一次不等式（定理 28.5、方法 28.6–28.7）： 直线 将平面划分为两个半平面。边界绘制规则——含等号画实线、不含等号画虚线——与 Ch11 数轴端点的“实心点 vs 空心圆”规则一致。选点测试法（取原点）是用以判定所求半平面的基本方法。

  4. 联立二元一次不等式（定理 28.6–28.7、方法 28.8）： 不等式组的解为各半平面的交集——即可行域，必为凸多边形区域。绘制可行域需依次画各边界直线、定各半平面方向、取交集标出。

  5. 线性规划（定理 28.8、方法 28.9）： 在约束条件定义的可行域上求目标函数 的最值。顶点原理是核心——若可行域为有界闭凸多边形，最优解必在顶点处取得。解题步骤依次为：设变量 列约束 写目标 画可行域 求顶点 代目标 比较得结论。几何上，目标函数对应一族平行线——求最值即相当于将此直线沿可行域平移，寻找首次/最后接触点。

  1. “且”与“或”的区分是首要难点： 学生容易混淆不等式组（默认“且”，求交集）与分类讨论（使用“或”，求并集）。建议在引入时直接以集合语言建立框架——“且 ，或 ”——并以具体数字及数轴反复演示。一元二次不等式 的解为 或 （两个区间的并集），而联立不等式组 的解为 （一个区间，即交集）。这组对比例可帮助学生体会“两个区间的并集”与“两个区间的交集”之别。

  2. 一元二次不等式——先图后代： 建议教学顺序为：先教图解法（利用已学的抛物线图像），让学生建立“ 即图像在 轴上方”的几何直觉，再引入因式分解的符号表分析法。两种方法并行教授，让学生理解它们是同一事物的不同视角。对于 的恒正/恒负情况，建议以具体数字验证（例如 对所有 成立），避免死记结论。

  3. 边界的实线/虚线——DSE 高频扣分点： 二元一次不等式的作图题中，实线（含等号）与虚线（不含等号）的区分是最频繁的扣分项目。建议在课堂练习中反复强调，并设计“给定不等式 判断边界应为实线还是虚线”的简短速练。可类比 Ch11 数轴上的空心圆 与实心点 。

  4. 选点测试——以原点为默认首选： 除非边界直线通过原点，否则一律建议学生以 为测试点。代入 后不等式化为 （或 等），极易判断。若原点恰在直线上，可选 或 （亦易计算）。

  5. 线性规划的“翻译”能力——文字转数学模型： DSE 线性规划题目通常以长文字题形式出现，学生的首要障碍往往是将文字中的限制条件转化为不等式。建议系统训练以下常见的文字-数学转化：

    文字描述 数学表达
    “ 不少于 ”
    “ 最多为 ”
    “总数不超过 ”
    “ 的两倍加 至少为 ”
    “非负条件”

  6. 线性规划——顶点法 vs 平移法： 教材以顶点法（方法 28.9 步骤 6–7）为标准解题步骤——求各顶点坐标后代入比较，最稳妥且不易出错。但建议在讲解时辅以平移法的几何直观（推论 28.3），让学生理解“为什么最优解在顶点”而非机械记忆步骤。若有条件，使用动态几何软件（如 GeoGebra）演示直线族平移的过程效果极佳。

  7. 整数解问题——线性规划的常见变体： 当题目要求 为整数（例如“生产整数件产品”）而最优顶点非整数时，学生需检查邻近的整数格点。建议提醒学生：不可简单地四舍五入——须逐一检查附近的整数点是否仍在可行域内，再比较目标函数值。此为 DSE 常见考法。
